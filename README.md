# Resume Builder

A sleek and customizable **Resume Builder Web Application** built with **Flask, HTML, CSS, JavaScript, and Bootstrap**.
This application allows users to quickly generate professional resumes using multiple templates and download them as PDF files.

---

## 🌐 Live Demo

👉 https://resume-builder-9fg3.onrender.com

---

## 🚀 Features

* ✅ **Multiple Resume Templates**
  Choose from different resume layouts such as professional, modern, and minimalist styles.

* ✍️ **Form-Based Resume Creation**
  Users can easily fill in personal details, education, work experience, and skills through a simple form.

* 🖨️ **Export Resume as PDF**
  Download the generated resume as a PDF file.

* 🎨 **Responsive UI**
  Built using **Bootstrap** for a clean and mobile-friendly design.

* ⚙️ **Customizable Sections**
  Add or modify sections like:

  * Education
  * Work Experience
  * Skills
  * Certifications
  * Projects

## ⚙️ Installation & Setup

Follow these steps to run the project locally.

### 1️⃣ Clone the repository

```
git clone https://github.com/yourusername/Resume-Builder.git
```

### 2️⃣ Navigate to project directory

```
cd Resume-Builder
```

### 3️⃣ Install dependencies

```
pip install -r requirements.txt
```

### 4️⃣ Run the Flask application

```
python app.py
```

### 5️⃣ Open in browser

```
http://127.0.0.1:5000
```
